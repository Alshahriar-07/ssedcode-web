Features

Everything a terminal-native developer needs

🖥 Interactive CLI Simulator

Experience Seed Code before downloading.

A full terminal simulation in your browser — boot sequence, real commands, natural-language questions, themes, even matrix mode. No install, no account, 100% offline.

Launch →

Modern CLI

A thoughtfully designed command-line interface with crisp layouts, subtle color, and zero clutter.

Streaming Responses

Tokens render the instant they arrive. Watch answers form in real time with no waiting spinner.

OpenRouter Support

Access hundreds of models through OpenRouter with a single API key, including free-tier models.

Ollama — Fully Local

Run models on your own machine with Ollama — private, offline-friendly, and no API key required.

FreeModel & AeroLink

Start instantly with FreeModel's zero-billing setup, or connect through AeroLink for low-latency regional endpoints.

Markdown Rendering

Rich Markdown rendered directly in your terminal — headings, lists, tables, and links included.

Syntax Highlighting

Code blocks are highlighted for dozens of languages, so answers read like your editor.

Cross Platform

One tool, everywhere you work. Native support for Windows, Linux, and macOS.

Fast Startup

Cold start in milliseconds. Seed Code is ready before your terminal finishes drawing the prompt.

Configuration System

A simple, versionable config file for providers, models, themes, and keybindings — with a guided wizard.

Open Source

MIT licensed and developed in the open. Read the source, file issues, and ship improvements.

Beautiful Terminal UX

Every pixel of output is considered — from progress states to panels to the blinking cursor.
